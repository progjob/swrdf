
Результат:
	Скрипт, который выводит в консоль путь к используемым файлам.

Задача:
	Frida модуль для отслеживания файлов к которым обращается приложение во время работы:
		чтение/запись в файлы.
		определение существования файла в системе.

Описание:
	Постараться сделать так, чтобы общее время работы над заданием не превышало 8-12 часов

	Постараться определить некоторую "единую" точку входа для работы с файлами.
	В случае невозможности осуществить перехват нескольких функций по работе с файлами.

	Определить конкретное время RnD по определению "единой точки входа" для работы с файлами
